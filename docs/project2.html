Mine Navigation Search and Rescue.

Embedded C for Microcontrollers, Paired Project (Imperial College London)

Project Overview

Our task was to develop an autonomous robot that can navigate a "mine" using a series of instructions coded in coloured cards and return to its starting position.
The buggy was to be able to navigate towards a coloured card and stop before impacting the card, read the card colour, interpret the card colour using a predefined code and perform the navigation instruction, navigate back to the starting position, handle exceptions and return back to the starting position if final card cannot be found.
